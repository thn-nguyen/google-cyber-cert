## Google Cybersecurity Professional Certificate

### Purpose
This repository includes completed hands-on activities as part of the Google Cybersecurity Professional Certificate. It serves as a collection of practical labs and exercises completed throughout the course. These activities demonstrate foundational cybersecurity concepts, tools, and workflows applied in real-world scenarios.

---

### Current Status
The activities are completed, but summaries are still in progress. This repository will continue to be updated in the future.

---

### Repository Structure
Each folder contains a separate activity with relevant files, notes, or documentation explaining the task and learning outcomes.
